# A1 Demo Page

Final expression:

$$
\mathrm{FSR} = \frac{c}{n_g L}
$$

??? note "Derivation (click to expand)"
    Start from the resonance condition:

    $$
    \beta(\omega) L = 2\pi m
    $$

    Differentiate w.r.t. $\omega$:

    $$
    \frac{d\beta}{d\omega} L = 2\pi \frac{dm}{d\omega}
    $$

    Using $n_g = c \frac{d\beta}{d\omega}$, we get:

    $$
    \mathrm{FSR} = \frac{c}{n_g L}.
    $$
